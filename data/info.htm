Parameter
Color

The color list is used for some of the effects (STATIC, FADE, METEOR) to define the LED colors. It has to contain a comma separated list of HEX color values (e.g. ffffff,00ff00,000000). If less values are set than LEDS are available the remaining leds are set to the last color. This allows to set all LEDs to s asingle color by simply passing a single value.

Speed

The speed of the animation. Valid values are between 1 and 19.
