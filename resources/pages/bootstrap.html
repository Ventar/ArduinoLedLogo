Configuration

Wireless
The MDNS name allows the connection to the logo via the address <name>.local. This will only work if the protocol is supported. On Windows you can install the Bonjour extension to enable support of this.
In case the logo is not able to connect to the configured Wifi it opens an access point to allow the configuration of a new WiFi connection. You need to connect to this network and go to IP address 192.168.4.1 and follow the instructions to select and configure a new network.
LEDs
The number of LEDs for the Logo.
The hardware PIN to which the LED strip is attached.
Animation
The delay configured as 100% for the animation speed. Increasing this value will make all animations slower, decreasing it will make them faster.
The animation default speed for the program tab between 0% and 100%
Buttons
The comma separated ist of pins used to controll buttons.
